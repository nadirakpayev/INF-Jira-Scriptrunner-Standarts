## Как использовать пакет

1. Начинайте с [00_стандарт_разработки_на_ScriptRunner_v2.md](00_стандарт_разработки_на_ScriptRunner_v2.md).
2. Для правил хранения и изменения переходите в `Хранение и версионирование`.
3. Для оформления страницы скрипта переходите в `Стандарты документации`.
4. Для написания кода переходите в `Стандарты разработки`.
5. Для логов и диагностики переходите в `Логирование`.
6. Для проектирования решения переходите в `Архитектура и переиспользование`.
7. Для ревью и обучения переходите в `Паттерны и анти-паттерны`.
